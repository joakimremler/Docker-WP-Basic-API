# Docker-WP-Basic-API

## Run Docker
docker-compose up -d

## Clean WP and add Basic Api Theme
bash setup.sh

## Stop all Docker processes
bash stop.sh

## Visit your sites:
Wordpress installation
http://localhost:80

phpadmin
http://localhost:8080

Mysql folder (Not Mac)
/var/lib/mysql

## Activate plugins in Wordpress
http://localhost/wp-admin/plugins.php
